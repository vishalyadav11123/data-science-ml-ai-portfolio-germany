# Project Notes

## What I Built

I created an interactive Power BI dashboard showing Germany’s economic snapshot from 2011 to 2024.

## Main Dashboard Sections

- KPI cards
- Trend line charts
- Benchmark bar chart
- Trade comparison chart
- Latest indicator table

## Challenges Faced

- Converting World Bank wide-format data into long format
- Fixing percentage formatting in Power BI
- Managing slicer interactions
- Shortening long World Bank indicator names
- Designing a clean dashboard layout in Power BI Online
